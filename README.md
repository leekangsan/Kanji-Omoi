# - Kanji-Omoi - 일본 상용 한자 2136자 외우기 프로젝트
- 일본 상용한자 2136자를 랜덤 깜빡이 형태로 표시해주는 프로젝트입니다.


# - Why Make this ?
- 방대한 양의 한자를 눈으로라도 쉽게 외우기 위해 깜빡이 형태로 한자를 외우기 쉽게하기 위해서 이 프로젝트를 시작하였습니다.
